# My Library 2
 My Library allows users to create nested categories. Every category have cards which contains notes, images or files.
 
 Imprtant services and packages:
  - Firebase Firestore
  - Firebase Authentication
  - Firebase Storage
  - Riverpod
  - AutoRoute
